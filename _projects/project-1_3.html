--- title: "Gravity Compensation Using Bernstein Polynomial Approximation" excerpt: | Precise gravity compensation for preventing unintended tool movement caused by gravitational forces in admittance control mode.
collection: projects --- This project implements Bernstein polynomial approximation to achieve high-precision gravity compensation for SHER-3.0. The goal is to eliminate unwanted tool drift or movement resulting from gravitational forces, particularly in systems operating under admittance control. By accurately modeling the gravity-induced torque using Bernstein polynomials, the control system ensures more stable tool behavior, contributing to safer and more predictable interaction during delicate tasks such as retinal surgery.
